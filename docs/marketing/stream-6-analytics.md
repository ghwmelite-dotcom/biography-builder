# Stream 6: Analytics & Optimisation

## 6A. Free Analytics Setup

### GA4 Custom Events

| Event Name | Trigger | Why It Matters |
|---|---|---|
| brochure_started | User begins creating a brochure | Measures intent |
| brochure_completed | User finishes and downloads/shares | Measures conversion |
| memorial_page_created | Memorial page goes live | Core product usage |
| memorial_page_shared | User clicks share | Viral loop tracking |
| obituary_created | Obituary page published | Product adoption |
| budget_planner_used | User completes budget planner | Feature engagement |
| qr_code_generated | QR code created | Feature engagement |
| signup_completed | New account created | Growth metric |
| referral_link_shared | User shares referral link | Viral coefficient |
| theme_selected | User picks a theme | UX insight |

### Google Search Console Monitoring

| Frequency | Check | Action |
|---|---|---|
| Daily (month 1) | Indexing status | Submit sitemaps, fix crawl errors |
| Weekly | Top queries, impressions, clicks | Optimise page 2 rankings |
| Weekly | CTR by query | Rewrite low-CTR title tags/meta descriptions |
| Bi-weekly | Core Web Vitals | Fix performance issues |
| Monthly | Backlink profile | Reach out for more links |
| Monthly | Declining pages | Update content, add internal links |

### UTM Naming Conventions

| Parameter | Convention | Examples |
|---|---|---|
| utm_source | Platform, lowercase | twitter, whatsapp, facebook, email |
| utm_medium | Content type | social, referral, email, bio, qrcode |
| utm_campaign | Campaign name, hyphens | launch-march-2026, funeral-planning-guide |
| utm_content | Specific variation | thread-1, carousel-costs, dm-churches |

## 6C. Monthly Review Framework

### Checklist

- [ ] Traffic: compare to previous month, which channels grew/declined?
- [ ] Conversions: sign-ups, brochures, memorials. Rate from visitor to sign-up?
- [ ] Top content: which 3 posts drove most traffic and conversions?
- [ ] SEO: how many keywords on page 1 vs page 2?
- [ ] Social: follower growth, engagement rate, best post
- [ ] Referrals: how many? viral coefficient?
- [ ] User feedback: recurring requests or praise?
- [ ] Partnerships: which generated results?
- [ ] Competitors: anything new?

### When to Pivot

| Signal | Action |
|---|---|
| Zero engagement after 30 posts | Pause 2 weeks. If no effect, abandon. |
| Traffic but zero conversions | Check landing page. Fix or abandon. |
| Time 10x exceeds results vs another channel | Reallocate. |
